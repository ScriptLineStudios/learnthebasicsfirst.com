Learn the Basics First

bluecap3: I want to re-create Jarvis from Iron Man.
bluecap3: I want to make a first person shooter with multiple endings.
These are multiple messages that are asked every now and then in online chat rooms; it's great to see you have enthusiasm to start such a big project, but it's better to start small.

Starting on a project would not be super difficult if you have a solid understanding of the basics. However, to someone who has never programmed before will be a huge challenge.

Starting with small projects such as Tic-Tac-Toe, Wordle, or Pong can help you build your foundational knowledge so you can move up to larger projects. Learn a little at a time.
